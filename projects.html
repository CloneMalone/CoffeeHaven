Class Projects

Tokyo Travel Guide

Tokyo Travel Guide is a simple website that guides the user on a Tokyo Japan experience so that they know what to expect when they visit. This website taught me how to make responsive HTML tables, and how to integrate video into a page. Click the video to check it out!

Shape Museum

Shape Museum is an A-Frame project that allows the user to move their first person figure around with their pc, or in a VR environment. This project gave me the necessary skills to create a functioning VR environment. Click the video to check it out!

Virtual Reality Website

The Virtual Reality Website is a resource for users to learn about VR history and also review a few VR products such as the Meta Quest 3 and the Valve Index. This project taught me how to make responsive websites that collapse and expand depending on the viewport. Click the image to check it out!

Porfolio Website

This is a simple HTML/CSS website with a home page and a link to my introduction project. This project taught me how to use basic HTML and CSS principles to build a simple website. Click the image to check it out!
